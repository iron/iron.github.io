fn write_headers(&mut self) -> Result<(), IoError>

Write the Status-Line and headers of the response, in preparation for writing the body.

This also overrides the value of the Transfer-Encoding header (self.headers.transfer_encoding), ensuring it is None if the Content-Length header has been specified, or to chunked if it has not, thus switching to the chunked coding.

If the headers have already been written, this will fail. See also try_write_headers.
